pytket-quest
==================================

`QuEST <https://quest.qtechtheory.org/docs/>`_ is a high performance simulator of quantum circuits, 
state-vectors and density matrices.

``pytket-quest`` is an extension to ``pytket`` that allows ``pytket`` circuits
to be simulated using ``quest``.

``pytket-quest`` is available for Python 3.10 and 3.11 on Linux, MacOS and
Windows. To install, run:

::
   
   pip install pytket-quest
